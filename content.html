INTRODUCTION

LA création de page web est fonction du language utilisé. Pour ce cas, nous utiliserons le language HTML qui repose sur quelques concepts très différents de ceux que l'on dans un traitement de texte tel que word.

DEFINITION

HTML: Hyper Text Markup Language. C'est un laguage de balisage perttantant de structurer le contenu d'une page web.
Page Web: est un document HTML pouvant les texts; audios; videos; images et hyperliens.
Balise: chaine de caratère permettant d'indiquer au navigateur comment il doit interpreter la page web.
Site web: ensemble de page web relié par les hyperliens.
URL: Unifrom Ressource Locator, c'est le chemin d'accès à une ressource sur le web (www).
Attribut: est une information qui renseigne sur les détailles d'interprétation d'une balise.
Ses avantages sont:
  • Peux coûteux en effet un simple éditeur de texte suffit à ecrire se prmiers documents html
  • Relativement facile à aborder
  • il représente en outre un bon moyen de dépasser les problèmes de comptabilité entre les systèmes et des formats infomatiques différents.
Ses limites sont:
  • Pas de confidentialité du code;
  • Il est incompatible entre les navigateurs;

LES OUTILS LOGICIELS DU HTML

Navigateur est un logice d'application tournant sur une machine cliente capable d'interroger les serveurs web, d'exploiter leurs résultats et de les affichers grâce aux balises contenues dans les pages HTML. ou encor un logicel d'application capable d'interpreter et d'afficher les documents web. Exemple: Mozilla Firefox; Internet Explorar; Netscape Navigator; Safari; Opéra; Google chrome; Aurora; Uc browser.
Un éditeur de texte est un programme dédié à l'écriture des codes. On peut en général l'utiler pour des multiples languages au delà du HTML. Exemple: Sublime text; Notepad ++; Bloc-note; DreamWear; Netscape Composer; Webfast; Adobe Golive; WebExpert;Nano WebEditor;FrontPage; HotMetal; Adobe PageMill; ScriptEdit.

LES HYPERS LIENS

Ils existent trois type de liens:

Lien absolue

Il va du repertoire racine courant jusqu'au dossier que vous vous trouvez. Illustration <a href="C:\Users\pc\Desktop\cours\cont.html"> ; cependant, il est moin utilisé.

lien relatif

Il est finie par la balise <a>. Sa syntaxe est la suivante <a href="url">placer votre titre<a>. EXEMPLE

les ancres

Se sont les liens appelés à l'intérieur des pages. Sa syntaxe est la suivante <a href="#id">placer votre titre<a>. EXEMPLE
L'attribut target est est utilisé pour définir la façon selon laquelle le lien doit être ouvert. Elle peut prendre les valeurs suivantes:
  • _blank - Ouvre le lien du document dans une nouvellle fenêtre.
  • _self - Ouvre le lien du document dans la même fenêtre (par default).
  • _parent - Ouvre le lien du document dans le frame parent.
  • _top - Ouvre le lien du document dans le plein corp de la fenêtre.
  • framename - Ouvre le lien du document dans le nom du frame

LES TABLEAUX

Un tableau est insérer par la balise <table>;chaques lignes du tableau sont précisées par <tr>. Dans une ligne une cellule est insérer avec <td>. EXEMPLE

REMARQUE: par défaut, les bordures du tableau ne sont pas représantées. l'attribut boder dans la balise <table> permet d'indiquer l'épaiseur des bordures.
<thead> permet d'indiquer l'entête du tableau.
<tbody> permet d'indiquer le corps du tableau.
<tfoot> permet d'indiquer le pied du tableau.
<th> remplace la balise <td> et permet d'indiquer une cellule d'enête. Par defaut les celleules <thead> sont interpreter en gras.
IL est possible les lignes et les colonnes:
pour les lignes, il faut préciser a l'intérieur de la cellule le nombre de ligne occupé par le cellule à l'aide de l'attribut rowspan.
Quand au colonne, elle est précisé à l'intérier de la cellule le nombre de colonne occupé par cette cellule à l'aide de l'attribut colspan
<caption< permet de donner un titre à un tableau. Il doit toujours être placé directement après <table>.EXEMPLE

LES IMAGES

L'image est insérer grâce à la balise <img>; l'attribut src permet de préciser l'url de l'image; alt pour la description de l'image s'il est ommis, l'image ne s'affichera pas; width et heigth permet de préciser la longueur et la hauteur de l'image son unité peut être en pixel; em; pourcentage;. Sa syntaxe est la suivante: <img src="url" alt="nom de l'image" /> . EXEMPLE

Remaque: Il est possible d'appeller les images dans autre dossier. Il s'effectue comme suit <img src="/dossier/url" alt="nom de l'image" />.
<picture> permet de définir le dégré de persection des images sur plusieurs appareils; la balise <source> permet de définir la nature de l'image grâce aux attribut media et srcset EXEMPLE

<map> qui contient toute les autres l'attribut name est associé avec <img> pour former une reletion et rendre clickcapable. EXEMPLE

Pour inser une image qui est contenue dans un autre dosier, préciser dans l'attribut src le non du repertoire puis un lash et efin l'url de l'image. quant au image contenue dans les serveurs distants, preciser dans l'attribut src son url. Ils est possible d'inserer les images qui ont l'extention .gif
